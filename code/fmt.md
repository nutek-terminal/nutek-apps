# fmt

## Description

fmt is a small, safe and fast formatting library providing a fast and safe alternative to C printf and C++ iostreams.

## Installation

```bash
brew install fmt
```

## Usage

```c++
std::string s = fmt::format("The answer is {}.", 42);
```
